# To-Do Task App

A simple and interactive To-Do List application built with HTML, Tailwind CSS, and Vanilla JavaScript.

## Features

- Add new tasks
- Update existing tasks
- Delete tasks
- Dynamic task rendering
- Responsive UI
- Simple CRUD functionality

## Technologies Used

- HTML5
- Tailwind CSS
- JavaScript (Vanilla JS)

## Project Structure

```bash
📦 to-do-task
 ┣ 📄 index.html
 ┣ 📄 script.js
 ┣ 📄 README.md
```

## How It Works

This project allows users to manage tasks dynamically using JavaScript DOM manipulation.

### CRUD Operations

- **Create:** Add a new task
- **Read:** Display tasks on screen
- **Update:** Edit task content
- **Delete:** Remove tasks from the list

## Installation

1. Clone the repository

```bash
git clone https://github.com/jhonata0200p/to-do-task.git
```

2. Open the project folder

```bash
cd to-do-task
```

3. Run the project

Open `index.html` in your browser.
